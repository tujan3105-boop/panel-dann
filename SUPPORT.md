# Support

Thanks for using GantengDann. This project is community supported.

## Where to Get Help
- Questions and how-to: start a discussion (if enabled) or open a Q&A issue.
- Bugs: open an issue using the bug report template.
- Feature requests: open an issue using the feature request template.

## Security
For security vulnerabilities, follow `SECURITY.md` and avoid public disclosure.

## Expectations
Support is best-effort. Clear reproduction steps and logs help us respond faster.
